COURS INTERACTIF IST — VERSION DÉCOUPÉE

Contenu :
- projet Vite React complet
- src/App.jsx : composant principal léger
- src/data/chapters.js : tout le contenu pédagogique
- src/components/*.jsx : composants séparés
- src/components/diagrams.jsx : schémas SVG séparés
- src/index.css : styles sans Tailwind

Installation :
1. Ouvrir un terminal dans le dossier du projet
2. npm install
3. npm run dev

Cette version est pensée pour être plus stable qu’un gros fichier JSX unique dans canvas.
